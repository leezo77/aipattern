The OutSystems Intelligent Automation Playbook

A guide to implementing AI Agentic Patterns with OutSystems Workflows, AI Agent Builder, and Agent Workbench.

OutSystems Implementation Framework

When to use Workflow

Use for structured, deterministic business processes. Ideal for orchestrating a sequence of automated tasks and human approvals where the rules are clear and the path is predictable.

When to use AI Agent Builder

Use for creating simple, no-code conversational agents. Perfect for RAG use cases where the agent's primary function is to answer questions based on a specific set of documents or knowledge base.

When to use Agent Workbench

Use for complex, low-code agents that require custom reasoning, tool use, and orchestration. The choice for building autonomous agents that can perform actions and collaborate with other agents.
